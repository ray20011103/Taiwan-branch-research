# 🎯 Project Pitch: 台股籌碼 Smart BPS 系統
> **"Uncovering the Invisible Hand in the Market."**
> 利用機器學習與大數據，精準捕捉營收公告前的「主力偷跑」訊號。

---

## 1. 核心痛點 (The Problem)
在台股市場，**「籌碼」**是領先指標，但數據充滿雜訊：
- **雜訊過多**：每日數十萬筆分點交易，90% 來自無參考價值的散戶與隔日沖。
- **主力隱形**：真正的波段主力往往隱藏在大量交易中，難以用肉眼識別。
- **資訊不對稱**：內線與知情人士往往在營收公告前 5-10 天就已完成佈局。

## 2. 解決方案 (The Solution)
我們構建了一套 **「行為金融量化系統」**：
1.  **行為特徵識別 (Identify)**：利用 **K-Means + PCA**，根據交易行為（留倉比、頻率、週轉率）將券商標籤化，鎖定具備核心主力的 **「聰明錢 Cluster」**。
2.  **動態訊號追蹤 (Follow)**：計算 **Smart BPS (Broker Prediction Signal)** —— 即該 Cluster 成員在當下的**淨買超張數**。
3.  **預測與埋伏 (Predict)**：在利多公告前，偵測 Smart BPS 的異常放量，實現提前佈局。

---

## 3. 技術亮點 (Secret Sauce)
本專案的核心在於**「行為風格分析」**而非單純的歷史損益：
*   **超越獲利排名**：傳統策略只看誰賺錢（Winners），本系統看誰具備**「主力風格」**（高留倉、高集中度、低頻率），有效過濾運氣成分與當沖噪訊。
*   **動態適應 (Adaptive Rolling Window)**：採用 **60日滾動視窗**，即時捕捉主力行為改變 (Regime Change)。
*   **穩健特徵 (Robust Feature Engineering)**：使用 **RobustScaler** 與 **PCA 降維**，有效處理金融數據的長尾分佈與離群值。
*   **視覺化驗證 (UMAP Visualization)**：將高維數據降至 2D 平面，直觀展示「主力」與「散戶」在行為空間上的物理距離。

---

## 4. 實證績效 (Traction - 2025 H1)
針對市場最熱絡的 16 檔標的進行 A/B Test 回測：

| 指標 | Smart BPS (本策略) | Original BPS (全市場) |
| :--- | :---: | :---: |
| **交易策略** | Long Only + 7% Stop Loss | 同左 |
| **總報酬** | **+192%** | +200% |
| **夏普比率** | **0.82 (優)** | 0.76 |
| **代表作** | **6215 和椿 (+71%)**<br>**3706 神達 (+25%)** | (受雜訊干擾較多) |

> **結論**：Smart BPS 雖然在強多頭中總獲利略低，但**風險調整後報酬 (Sharpe)** 顯著較高，能提供更穩健的投資回報。

---

## 5. 應用場景 (Use Cases)
*   **每月營收埋伏**：在每月 1-10 號前，掃描 Smart BPS 異常買進標的。
*   **法說會/財報押寶**：偵測特定事件前的知情交易。
*   **空頭市場防禦**：當全市場恐慌時，Smart BPS 能偵測到主力低檔承接的訊號。
