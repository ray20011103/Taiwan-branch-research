第三層：分點券商的「元特徵」 (Broker Branch Meta-Features) —— 這才是真正的 Alpha

這是目前比較少人深入做，但含金量最高的地方。與其看「哪家券商買最多」，不如分析「買進的人是誰」。

1. 地緣券商因子 (Geographic Proximity)

很多中小型公司的消息會先從公司內部傳出，員工或親友往往透過公司附近的券商下單。

實作方法：

利用 Google Maps API 或爬蟲取得上市櫃公司總部地址。

計算所有券商分點與該公司總部的物理距離。

因子構建： 統計「距離該公司 5 公里內的券商分點」之淨買賣超總和。

Hypothesis: 如果地緣券商在股價無波動時突然大買，通常是內線或利多的前兆。

2. 券商獲利能力評分 (Broker Profitability Score, BPS)

這是一個動態特徵。不要假設「高盛」就是聰明錢，我們要看數據說話。

邏輯： 找出「過去一段時間在這檔股票上賺最多錢的分點」，跟隨他們的動作。

回測實作步驟：

回溯 (Lookback)： 在 t 日，計算過去 60 天，每一家分點券商在該股票上的「已實現+未實現損益」。

排名 (Ranking)： 挑選出獲利最高的前 5 名分點（贏家圈）。

訊號 (Signal)： 觀察這 5 家贏家券商在 t 日是買進還是賣出。

因子值： ∑(Top5_Winners_NetBuy)。如果贏家都在買，跟單勝率極高。
